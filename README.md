## How to Install

1. Press the bright green code button and click on 'download as zip'
2. Un-zip the folder
3. Open Steam
4. Right Click on BMC, click manage, browse local files
5. Open Assets
6. Make a backup of data.jet (Optional)
7. Replace data.jet with modded data.jet
8. Open BMC
